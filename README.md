**# GitHub User Finder**



**Автор: Севостьянова Виктория Юрьевна**



**## Описание**



**GUI-приложение на Python (Tkinter) для поиска пользователей GitHub через GitHub REST API.**  

**Позволяет находить пользователей по имени, просматривать результаты и добавлять выбранных пользователей в избранное.**  

**Список избранных сохраняется в файл `favorites.json` в формате JSON.**



**## Как использовать API**



**Приложение использует GitHub Search API для поиска пользователей:**



**- Базовый URL: `https://api.github.com/search/users`**

**- Параметр `q` — строка запроса (имя пользователя или часть имени).**



**Пример запроса (для пользователя `octocat`):**



**```text**

**https://api.github.com/search/users?q=octocat**

**```**



**Ответ API содержит поле `items` — список найденных пользователей, где для каждого пользователя есть, в частности, поля `login` и `html\_url`.**

**1. Установите Python 3.**

**2. Клонируйте репозиторий:**



&#x20;  **```bash**

&#x20;  **git clone <URL\_ВАШЕГО\_РЕПОЗИТОРИЯ>**

&#x20;  **cd github\_user\_finder**

&#x20;  **```**



**3. Приложение использует только стандартные библиотеки Python (`tkinter`, `json`, `urllib`, `ssl`), дополнительные пакеты устанавливать не нужно.**



**4. Запустите приложение:**



&#x20;  **```bash**

&#x20;  **python main.py**

&#x20;  **```**



**## Как пользоваться**



**1. Введите имя пользователя или строку поиска в верхнем поле.**

**2. Нажмите кнопку \*\*Search\*\*.**

**3. В левой панели появятся найденные пользователи в формате `login | url`.**

**4. Выберите пользователя и нажмите \*\*Add to favorites\*\*, чтобы добавить его в избранное.**

**5. В правой панели отображается список избранных пользователей.**  

&#x20;  **Чтобы удалить пользователя из избранного, выделите его и нажмите \*\*Remove selected\*\*.**



**Все избранные пользователи сохраняются в файле `favorites.json` в корне проекта.**



**## Примеры тестов (по желанию)**



**- \*\*Пустой ввод\*\* — приложение должно показать сообщение об ошибке и не отправлять запрос.**

**- \*\*Поиск существующего пользователя\*\* (например, `octocat`) — должен появиться хотя бы один результат.**

**- \*\*Повторное добавление в избранное\*\* — во второй раз приложение сообщит, что пользователь уже есть в избранном.**

**- \*\*Удаление из избранного\*\* — после удаления пользователь исчезает из правого списка, а `favorites.json` обновляется.**
